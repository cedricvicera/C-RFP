## Usage

### Deployment
Requires Flask to be installed.
Install and update using [pip](https://pip.pypa.io/en/stable/quickstart/):

````
$ pip install -U Flask
````

Run the following script and click the link:

````
$ python apprunner.py
  * Running on http://127.0.0.1:5000/ (Press CTRL+C to quit)
````
